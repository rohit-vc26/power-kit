# Email Writing Rules — Power-Kit

**Every email you write must pass these 8 checks.** Power-Kit auto-checks them.

---

## The 8 Rules

### Rule 1: Sound Human (Not AI)
- Use contractions ("you'll", "we've", "don't")
- Mix short and long sentences
- Show personality with 1 unique phrase
- Reference real details (names, dates, places)

**BAD**: "Our solution provides comprehensive value optimization."  
**GOOD**: "We built this because our team wasted 3 hours a day on spreadsheets. You probably do too."

---

### Rule 2: NO Em-Dashes
The em-dash `—` is banned.

**BAD**: "This campaign — the one from last month — got great results."  
**GOOD**: "This campaign (the one from last month) got great results."

**Use instead:**
- Comma `,`
- Parentheses `()`
- Period `.`
- Hyphen `-` (en-dash, OK)

---

### Rule 3: 65-90 Words Per Section
**Why?**
- Under 65 words = cheap, low effort
- Over 90 words = too much to read
- 65-90 = just right

**How to check:**
- Use https://wordcounter.net
- Count main body only (not subject, not signature)

---

### Rule 4: CTA + Hook Below
Every email needs:

**Step 1:** Clear button/link  
`→ Click to get started`

**Step 2:** Hook below it (1-2 sentences)  
`P.S. Takes 2 minutes. No credit card needed.`

**Hook types** (pick one):
- Curiosity: "What's inside..."
- Social proof: "3,400 teams use this"
- Reassurance: "No credit card. No spam."
- FOMO: "Ends Friday."
- Benefit: "See how Brand X got 3x more clicks..."

---

### Rule 5: Follow Team Rules
- No API keys in copy
- No customer passwords in copy
- No internal team names without permission
- HTTPS links only
- Verify all data before sending

---

### Rule 6: NO Spam Words

**Banned** (Power-Kit blocks these):

```
guaranteed, guarantee, warranties, warranted
risk-free, no risk
limited time, act now, click here immediately
don't miss out, claim your, free money
one-time offer, exclusive, last chance
must work, can't fail, 100% success
miracle, amazing results, unbelievable
```

**Replace with:**
- "guaranteed" → "most see" / "research shows"
- "limited time" → "available this week" / "ending Friday"
- "act now" → "get started today" / "try it"
- "claim your" → "get your"
- "exclusive" → "just for you"

---

### Rule 7: Real Promises Only
**Test:** Would you say this face-to-face to a customer?

**BAD**: "You'll make $1M this year!"  
**GOOD**: "Some customers have made over $1M in their first year."

(Specific. Past tense. Not for everyone.)

---

### Rule 8: NO Guarantees
**Banned:**
- "Guarantee"
- "Guaranteed"
- "Warrant"
- "Ensures" (in the sense of promising)
- "100% success"

**Replace with:**
- "Most users see..."
- "Customers have gotten..."
- "Your team can..."
- "Based on data from 50K emails..."

---

## Pre-Send Checklist

Before clicking SEND, answer YES to:

```
☐ Sounds human (not AI)?
☐ No em-dashes (—)?
☐ 65-90 words?
☐ Has CTA + hook below?
☐ Follows team rules?
☐ No spam words?
☐ Real promises only?
☐ No guarantees?
```

If ANY is NO → Don't send. Fix it.

## Power-Kit Auto-Checks

When you ask Claude to write/edit an email:
1. Claude writes the email
2. Power-Kit checks all 8 rules
3. If any rule is broken → warning shown
4. You fix it
5. Send when all 8 pass

## Quick Fixes

### "My email is 95 words"
- Cut adjectives
- Combine sentences
- Remove "thank you for taking the time" type phrases

### "I have 'guaranteed' in my email"
- Find/replace with "research shows" or "most see"
- Or delete the sentence

### "My CTA has no hook"
- Add this below CTA: "P.S. [Curiosity / Social proof / Reassurance]"

### "It sounds like AI wrote it"
- Add a contraction
- Add a real detail (your team, your industry)
- Read aloud — does it sound like you?
