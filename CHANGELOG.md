## How updates work

Power-Kit checks for updates each time you open Claude Code.

When an update is ready, you'll see:
```
Power-Kit update available: v1.0.1
What's new: [list of changes]
Update now? (y/n)
```

Type `y` and press Enter. Done.

---

## Version policy

- `v1.0.0` — Major.Minor.Patch
- **Major (1.x.x)**: Big changes. Might need re-training.
- **Minor (x.1.x)**: New skill or feature. Backward-compatible.
- **Patch (x.x.1)**: Bug fix or small improvement.
